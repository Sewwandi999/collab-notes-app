# Collaborative Note-Taking App

A MERN stack collaborative note-taking web application built with MongoDB, Express, React, Node.js, and Tailwind CSS.

## Features

- JWT authentication
- Create, edit, delete notes
- Rich text editor
- Full-text search
- Collaborator management
- Responsive modern UI

## Setup

### Backend

cd server
npm install
npm run dev

### frontend
cd client
npm install
npm run dev

### Environment Variables
### Create server/.env and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret

You can use server/.env.example as a template.
